## 🚀 セットアップ手順

1. **環境準備**:
   ```bash
   # リポジトリのクローン
   git clone <repository-url>
   cd interdimensional-shop
   
   # 自動セットアップ実行
   ./setup.sh
   ```

2. **手動セットアップ**:
   ```bash
   # 依存関係インストール
   npm install
   
   # 環境変数設定
   cp .env.local.example .env.local
   # .env.local を編集
   
   # データベースセットアップ
   npm run prisma:generate
   npm run prisma:migrate
   npm run prisma:seed
   
   # 開発サーバー起動
   npm run dev
   ```
